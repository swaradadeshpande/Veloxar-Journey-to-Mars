# VELOXAR – Journey to Mars
**An Immersive Interstellar Storytelling Experience**

VELOXAR is a high-performance, interactive web journey that chronicles humanity's 55-million-kilometre odyssey from Earth to the Red Planet. Built for the "Frontend Odyssey" challenge, this project transforms static data into a cinematic narrative through cutting-edge frontend technologies.

## 🚀 Key Features
- **Cinematic Narrative:** A 7-stage story covering Launch, Deep Space, Approach, Landing, and Exploration.
- **Starfield Engine:** A custom-built Three.js WebGL background with 8,000+ stars and dynamic shooting stars.
- **Adaptive UI:** A custom-engineered cursor system and a glassmorphism-based navigation system.
- **Mission Analytics:** Real-time data visualization using Chart.js to display Mars' atmospheric composition and mission stats.
- **Interactive Telemetry:** Clickable flip-cards and hotspots that reveal deep-mission technical data.

## 🛠️ Technical Stack
- **Core:** HTML5, CSS3 (Modern Flexbox/Grid), JavaScript (ES6+)
- **Animation:** GSAP (GreenSock Animation Platform) & ScrollTrigger
- **3D Graphics:** Three.js (WebGL)
- **Charts:** Chart.js
- **Icons:** Bootstrap Icons

## 📱 Responsiveness
The experience is fully optimized for:
- Desktop (Ultra-wide support)
- Tablets (Fluid grid layouts)
- Mobile (Hamburger menu and touch-optimized interactions)

## 🔧 Installation
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. For the best experience, enable sound via the UI toggle.
